** 2-tier deployment **

Description: The Vehicle Registration System is a Java-based application designed to manage vehicle registration information stored in a MySQL database. Leveraging Git, GitHub, Docker, Docker-compose, Jenkins, Amazon ECR, Kubernetes, and Amazon EKS, this system provides a robust and scalable solution for vehicle registration processes.

1. Git & GitHub: Git is utilized for version control, enabling collaborative development and version tracking among team members GitHub serves as the remote repository hosting service, facilitating code sharing, collaboration, and project management. Developers can work on different features and fixes concurrently, managing changes through branches and pull requests.

2. Docker & Docker-compose: Docker is employed for containerization, allowing the application components to be encapsulated into lightweight, portable containers.

3. Docker-compose simplifies the management of multi-container Docker applications by defining and running multi-container Docker applications with a single YAML file. This approach ensures consistency across different environments and simplifies the deployment process.

4. Amazon ECR & Kubernetes (EKS): Amazon Elastic Container Registry (ECR) is used as a fully managed Docker container registry within the AWS ecosystem, ensuring secure storage and retrieval of Docker images for deployment. Amazon Elastic Kubernetes Service (EKS) serves as the orchestration platform for managing and scaling containerized applications within the AWS cloud environment.
